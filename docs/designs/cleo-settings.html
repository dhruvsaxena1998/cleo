Overview
CLEO is a terminal UI for discovering and managing tmux sessions across your projects with intelligent agent mapping. Changes made here are saved to ~/.config/cleo/config.toml.

How the Dashboard Works

  • ●Left panel lists all your projects and their active/idle session counts.
  • ●Center panel shows sessions grouped by agent for the selected project.
  • ●Right panel shows detailed info for the selected session (toggle with t).
  • ●Activity log at the bottom tracks all session events in real time.
  • ●Use / to search across sessions, agents, projects, or branches.

Workflow

  • ●Press Enter on any session to attach — you enter the tmux session directly.
  • ●Detach with Ctrl+B d (tmux detach) to return to CLEO. Your work keeps running.
  • ●Press N to create a new session for the selected project.
  • ●Sessions persist across CLEO restarts — tmux is the source of truth.
Tip: Settings changed here are written to the config file immediately. Some settings (like keybinds) require restarting CLEO to take effect.
